Сначала приложение пытается определить IP-адрес пользователя через request.headers.get('X-Forwarded-For', request.remote_addr). Если приложение запущено локально, то город устанавливается вручную как Барнаул. Если сайт открыт через интернет, выполняется запрос к сервису ip-api.com, который определяет город пользователя по IP-адресу. Потом вызывается функция get() из файла api.py, в которую передаётся город. Функция get() получает координаты города через Yandex Geocoder API, затем отправляет запрос в Yandex Weather API и возвращает текущую температуру, влажность воздуха и облачность. Полученные данные сохраняются в переменные city, lat, lon, temp, humidity, cloudiness.Если погоду получить не удалось, создаётся сообщение об ошибке и передаётся в HTML шаблон. Если данные успешно получены, Flask передаёт их в файл templates/index.html через функцию render_template(). В шаблон отправляются город, координаты, температура, влажность и облачность, после чего пользователь видит готовую страницу с погодой.
Про api.py get():
Сначала функция отправляет запрос к Yandex Geocoder API для получения координат города. Пользователь передаёт название города через аргумент sity, после чего API возвращает широту и долготу. После получения координат функция формирует запрос к Yandex Weather API. Через запрос отправляются координаты города, а в ответ приходит информация о текущей погоде.
Значение облачности переводится на русский язык с помощью библиотеки deep-translator и класса GoogleTranslator.
